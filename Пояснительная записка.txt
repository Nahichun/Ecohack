Приложение реализует прототип системы оперативного обнаружения зон скопления плавающего пластика по спутниковым данным Sentinel-2.
Пользователь задаёт координаты и радиус анализа.
Система автоматически:
Загружает снимки за последние 3 суток.
Вычисляет индекс FDI (Floating Debris Index).
Маскирует облачность по слою SCL.
Нормализует результат.
Отображает карту концентрации.

В случае недоступности API автоматически активируется демонстрационный режим, который имитирует пространственные скопления мусора. Это обеспечивает устойчивость работы приложения на презентациях и в условиях ограниченного интернет-соединения.

Индекс FDI выбран как специализированный спектральный показатель для выявления плавающих объектов в морской среде, так как он снижает влияние отражательной способности воды.

Установка библиотек: pip install streamlit numpy folium streamlit-folium sentinelhub

Запуск приложения: streamlit run app.py
